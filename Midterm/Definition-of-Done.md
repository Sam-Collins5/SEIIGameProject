## Definition of Done
For this project, a feature or task is considered done when it meets the following criteria:

    The feature is fully implemented and functional within the educational video game.
    The feature has been tested by team members and confirmed to work without critical bugs or crashes.
    The implementation supports the educational purpose of the game, allowing the player to progress through gameplay while interacting with learning challenges.
    The feature has been committed and pushed to the GitHub repository, and merged into the main branch if applicable.
    The code follows the team’s coding standards and project structure.
    Any required assets, scripts, and scene changes are properly integrated into the game.
    The feature does not introduce game-breaking bugs or regressions to existing mechanics.
    The feature has been reviewed by at least one team member and approved for inclusion in the project build.

# Project Goals
## The primary goals of this project are:
    Develop a fully playable educational video game.
    Implement a character that can move through different environments.
    Allow players to interact with challenge doors that trigger educational taacher challenges.
    Implement a question and answer system that affects gameplay progression.
    Create multiple rooms and scenes connected through doors.
    Ensure the game runs smoothly without major bugs or crashes.
    Have a ending to the game after the user defeats all teacher challenges.
